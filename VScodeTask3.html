VS Code

VS Code is an open-source code editor developed by Microsoft, primarily for web development. It provides tools for building applications with a graphical user interface (GUI). Though it's a code editor, with extensions, it functions like an Integrated Development Environment.(IDE).

Key features

  • Platform Support:Available on Windows, macOS, and Linux, allowing users to work seamlessly across different platforms.
  • Development Capabilities:Offers built-in Git integration, intelligent code completion (IntelliSense), syntax highlighting, debugging tools, and more.
  • Open Source:VS Code is free and open-source, meaning anyone can contribute to its development or customize it for specific needs.
  • Built-in Terminal:Terminal for executing commands within the editor.
  • Version Control:Integrated Git support for version control, branching, and merging.
  • Remote Development:Work on remote machines or containers with SSH, WSL, or Docker.

Why should we use VS Code?

  1. Efficiency:VS Code improves coding productivity with features like code completion, syntax highlighting, and integrated debugging.
  2. Cross-platform:It runs on Windows, macOS, and Linux, allowing developers to work on any system.
  3. customization:Users can tailor the editor according to their preferences using extensions, themes, and key bindings.
  4. Extensions and Integrations:VS Code supports a wide range of extensions, providing extra functionalities like version control, language support, and more.
  5. Lightweight and Fast:It is quick to install and consumes fewer resources compared to full IDEs.
  6. Community Support:Being open-source, it has a large community that contributes to its continuous improvement.

Extentions of VS Code

Extensions in VS Code are add-ons that enhance its functionality by supporting additional languages, frameworks, tools, and features. They provide language support with syntax highlighting and IntelliSense, enable code formatting (like Prettier), integrate version control tools such as Git, allow real-time live previews (e.g., Live Server), and offer customization options like themes and icons. Extensions make VS Code more powerful and tailored to meet specific development needs, improving efficiency and user experience.

Extension name usage
Live Server Launches a local server with live reload for static and dynamic web pages, ideal for web development
Live Preview Provides a live preview of your HTML, CSS, and Javascript files directly in the browser.
Prettier Automatically formats your code to ensure consistent style across the project(supports many languages)

ESLint Integrates ESLint into VS Code for identifying and fixing JavaScript code issues based on defined rules.

Emmets and their Abbreviations

Emmet is a powerful toolkit and abbreviation syntax used by web developers to speed up HTML and CSS coding. It allows for the shorthand entry of complex code structures, significantly improving productivity and reducing manual coding time.

Emmet Abbreviation
! Expands to the basic HTML structure, including the <!DOCTYPE>, <html>, <head>, <body>, and more.
+ Creates a sibling element. For example, div + p will create a div and a p next to each other.
> Creates a child element. For example, ul > li creates a ul element with a li child element.
* Repeats the element a specified number of times. For example, ul > li*5 creates a ul with 5 li items inside.
^ Moves the cursor to the parent element. For example, div > p^ moves the cursor to the parent div of p.
{ } Used to insert text or content inside an element. For example, div{Hello} creates a
